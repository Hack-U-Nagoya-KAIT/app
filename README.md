# *旅の行き先提案アプリ"cotabi"* 


![cotabi title](app/react-expo/assets/logo.png)
![cotabi demo](cotabi-demo.gif)

# 概要
使える時間、予算、旅のコンセプトを入力して検索ボタンをたっぷすると条件に合致した目的地を動で選び、目的地の住所や移動にかかる時間、交通費などが乗った旅のしおりを自動で作成する。
HACK U に参加時作成

# 機能
- 旅のしおりの生成
  - 現在地取得
  - 周辺施設の情報の取得
  - コンセプトより施設を選定
  - 所要時間（車）を算出
  - 目的地・タイムスケジュールをテキスト形式で出力
